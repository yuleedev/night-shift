# night-shift
in this game, you play as a guard (of an undisclosed location) using a flashlight to observe the premises. any figures conveniently dressed in a black beanie and black and white striped burglar costume should be reported right away! there seems to be quite a few other beings and things looming around, though, so make sure you don't catch the wrong culprit..

this is another game made for https://onekey.hackclub.com/ using unity and c#! all art and programming by me.
play it on itch [here!](https://yuleedev.itch.io/night-shift)

## how to play
this game only requires your spacebar. if your flashlight reveals a person that looks like a burglar, press space to detect them and move on to the next level! the levels will get progressively harder. the game ends if you run out of time or press space on something other than the robber.
